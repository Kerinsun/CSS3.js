如果你的浏览器支持@supports，这行字会是绿色

@supports (display: block) {
	/* 如果支持display: block,则下面指定的CSS会生效 */
	.supports {
		color: green;
	}
}

检测是否支持transform属性或者其带前缀版本

/* 由于本demo同时也加载了prefixfree组件，所以无需写前缀 */
@supports (transform:rotate(7deg)) {
	.supports-transform {
		transform: rotate(7deg);
	}
}

检测是否支持appearance属性或者其带前缀版本

/* 由于本demo同时也加载了prefixfree组件，所以无需写前缀 */
@supports (appearance:button) {
	.supports-appearance {
		appearance: button;
	}
}

检测是否不支持指定的CSS属性

@supports ( not ((position: fixed)){
	/* 如果不支持position: fixed */
	position: absolute;
}
